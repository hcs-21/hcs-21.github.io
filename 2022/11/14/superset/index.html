一、Superset 环境准备

1.1、pyenv 安装配置

pyenv ： Python 多版本管理器，使用户在系统里安装和管理多个 Python 版本

安装依赖环境

yum -y install gcc gcc-c++ zlib-devel bzip2-devel openssl-devel ncurses-devel sqlite-devel readline-devel tk-devel gdbm-devel libpcap-devel xz-devel python-pip

安装git

yum -y install git

安装pyenv

git clone https://github.com/pyenv/pyenv.git

报错

fatal: unable to access 'https://github.com/pyenv/pyenv.git/': TCP connection reset by peer

解决

yum install -y nss curl libcurl

报错

fatal: unable to access 'https://github.com/pyenv/pyenv.git/': Failed connect to github.com:443; Connection refused

解决

git config --global --unset http.proxy
git config --global --unset https.proxy

配置环境变量

echo 'export PATH="/home/superset/pyenv/bin:$PATH"' >> ~/.bash_profile

echo 'eval "$(pyenv init -)"' >> ~/.bash_profile


source ~/.bash_profile

验证pyenv

pyenv -v

1.2、安装 python3.6

pyenv 查看可安装版本python

pyenv install --list

pyenv查看已安装版本python

[superset@lagou122 ~]$ pyenv versions
* system (set by /home/superset/.pyenv/version)
##系统版本

安装python3.6.6

pyenv install -v 3.6.6

pyenv查看已安装版本python

[superset@lagou122 ~]$ pyenv versions
* system (set by /home/superset/.pyenv/version)
  3.6.6

1.3、配置管理python版本

  • pyenv global 全局配置
  • pyenv shell shell配置,之影响档当前会话
  • pyenv local 当前目录模式

python 版本全局配置改为 3.6.6

pyenv global 3.6.6
pyenv rehash

查看python版本

[superset@lagou122 ~]$ python -V
Python 3.6.6

查看python管理器版本

[superset@lagou122 ~]$ pyenv versions
  system
* 3.6.6 (set by /home/superset/.pyenv/version)

恢复python系统版本

pyenv local system

pyenv管理器卸载安装版本

pyenv uninstall 3.6.6

1.4、安装pyenv-virtualenv插件

克隆插件

git clone https://github.com/yyuu/pyenv-virtualenv.git ~/.pyenv/plugins/pyenv-virtualenv

添加环境变量

eval "$(pyenv virtualenv-init -)"

检查插件是否添加成功

pyenv help virtualenv

1.5、创建一个隔离环境

pyenv virtualenv 3.6.6 myenv

切换到虚拟环境

pyenv activate myenv

退出隔离环境

pyenv deactivate

删除隔离环境

rm -f /home/superset/.pyenv/versions/myenv

二、superSet安装配置(myenv)

2.1、安装依赖包

更新python-setuptools

sudo yum upgrade python-setuptools

安装依赖包

sudo yum install -y gcc gcc-c++ libffi-devel python-devel python-pip python-wheel openssl-devel cyrus-sasl-devel openldap-devel

可能遇见的问题

解决方案

sudo yum -y install epel-release

sudo yum -y install python-pip
sudo yum -y install python-wheel

2.2、指定国内源更新pip setuptools

pip install --upgrade setuptools pip -i https://pypi.douban.com/simple/

2.3、安装配置superSet

安装superSet

pip install apache-superset -i https://pypi.douban.com/simple/

初始化superSet数据库

superset db upgrade

可能遇到的错误

解决方案

pip install sqlalchemy==1.3.24

可能遇到的错误

解决方案

pip install dataclasses

创建管理员用户

##设置环境变量
export FLASK_APP=superset
##设置用户名和密码
superset fab create-admin

初始化superset

superset init

2.4、启停服务

安装 gunicorn

pip install gunicorn -i https://pypi.douban.com/simple/

启动superset

gunicorn --workers 5 --timeout 120 --bind lagou122:8787 "superset.app:create_app()"  --daemon

访问

lagou122:8787/login/

停止superset

ps -ef | awk '/superset/ && !/awk/{print $2}' | xargs kill -9

1、IO流的基本分类

读写单位：

  1. 字节流
  2. 字符流

读写方向：

  1. 输入流
  2. 输出流

读写角色：

  1. 节点流
  2. 处理流

2、体系结构

分类 字节输入流 字节输出流 字符输入流 字符输出流
抽象基类 InputStream OutputStream Reader Writer
访问文件 FileInputStream FileOutputStream FileReader FileWriter
访问数组 ByteArrayInputStream ByteArrayOutStream CharArrayReader CharArrayWriter
访问管道 PipedInputStream PipedOutputStream PipedReader PipedWriter
访问字符串 —— —— StringReader StringWriter
缓冲流 BufferedInputStream BufferedOutputStream BufferedReader BufferedWriter
转换流 —— —— InputStreamReader OutputStreamWriter
对象流 ObjectInputStream ObjectOutputStream —— ——
FileterInputStream FileterOutputStream FileterReader FileterWriter
打印流 —— PrintStream —— PrintWriter
推回输入流 PushbackInputStream —— PushbackReader ——
特殊流 DataInputStream DataOutputStream

3、相关流的详解

3.1、FileWriter类（重点）

基本概念：将文本内容写入到文本文件中

方法声明 功能介绍
FileWriter(String name) 根据参数指定的文件名构造对象
FileWriter(String name , boolean append) 以追加的方式根据参数指定的文件名构造对象
void write(int c) 写入单个字符
void write(char[] buf. int off,int len) 将指定字符数组中从偏移量off 开始的len个字符写入此文件输出流
void write(char[] cbuf) 将cbuf.length个字符从字符数组写入此文件输出流
void flush() 刷新流
void close() 关闭流对象，并释放所有资源

3.2、FileReader类（重点）

基本概念：主要用于将文本文件读取文本数据内容

方法声明 功能介绍
FileReader(String Reader) 根据参数指定的文件名构造对象
int read() 读取单个字符的数据并返回，返回-1标识读取到末尾
int read(char[] buff , int off , len) 从输入流中最多将len个字符读取到数组中，返回读取到的字符个数，返回-1表示读取到末尾
int read(char[] buff) 从输入流中最多将buff.length个字符读取到数组中，返回读取到的字符个数，返回-1表示读取到末尾
void close() 关闭对象流，并释放所有资源

3.3、FileOutputStream类（重点）

基本概念：主要用于将图像类的原始字节流写入到输出流中

方法声明 功能介绍
FileOutputStream( Stream name) 根据参数指定的文件名构造对象
FileOutputStream(Stream name,boolean append) 以追加的形式根据参数指定的文件名构造对象
void write(int b) 将指定字节写入此文件输出流
void write(byte[] b, int off,int len) 将指定字节数组中从off 开始的len个字节写入到此文件输出流中
void write(byte []) 将指定字节数组中byte.length个字节写入到此文件输出流中
void flush() 刷新此输出流并强制写出任何缓冲的输出字节
void close() 关闭对象流并释放有关资源

3.4、FileInputStream类（重点）

基本概念：主要用于从输入流中以字节流的方式读取图像类数据

方法声明 功能介绍
FileInputStream(String name) 根据参数指定的文件构造对象
int read() 从输入流中读取单个字节的数据并返回，返回-1表示读取到末尾
int read(byte[] b ,int off ,in len) 从此输入流中将最多len个字节的数据读入字节数组中，返回读取到的字节个数，返回-1表示读取到末尾
int read(byte[]) 从此输入流中将byte.length个字节的数据读入字节数组中，返回读取到的字节个数，返回-1表示读取到末尾
void close() 关闭对象流并释放有关资源
int available() 获取输入流所关联文件的大小

3.5、BufferedOutputStream类（重点）

基本概念：缓冲输出流，此时不用为写入的每个字节调用底层系统

方法声明 功能介绍
BufferedOutputStream(OutputStream out) 根据参数指定的引用构造对象
BufferedOutputStream(OutputStream out,int size) 根据参数指定的引用和缓冲区大小构造对象
void write(int b) 写入单个字节
void write(byte[] b , int off , int len) 写入字节数组中的一部分数据
void write(byte[]) 写入参数指定的整个字节数组
void flush() 刷新流
void close() 关闭流对象并释放所有资源

3.6、BufferedInputStream类（重点）

方法声明 功能介绍
BufferedInputStream(InputStream in) 根据参数指定的引用构造对象
BufferedInputStream(InputStream in , int size) 根据参数指定的引用和缓冲区的大小构造对象
int read() 读取单个字节
int read(byte[] b , int off , int len) 读取len 个字节
int read(byte[] b) 读取b.length个字节
void close() 关闭流对象并释放所有资源

3.7、BufferedWriter类（重点）

方法声明 功能介绍
BufferedWriter(Writer out) 根据参数指定的引用构造文件
BufferedWriter(Writer out , int size) 根据参数指定的文件和缓冲区大小构造对象
void writer(int c) 写入单个字符到输出流中
void writer(char[] cbuf,int off,int size) 将字符数组cbuff 中从下标off开始的len个字符写入到输出流中
void writer(char[] cbuf) 将字符数组cbuf 中所有内容写入到输出流中
void write(String s , int off , int len) 将参数S中从下标off 开始的len个字符写入到输出流中
void write(String str) 将参数指定的字符串写入到输出流中
void newLine() 用于写入分隔符到输出流中
void flush() 刷新流
void colse() 关闭流对象并释放所有资源

3.8、BufferedReader类（重点）

方法声明 功能介绍
BufferedReader(Reader in) 根据参数指定的引用构造对象
BufferedReader(Reader in , int size) 根据参数指定的引用和缓冲区大小构造对象
int read() 从输入流读取单个字符，读到末尾则返回-1，否则返回实际读取到的内容
int read(char[] cbuf,int off,int len) 从输入流中读取len 个字符放到数组cbuf 中off 开始的位置上，若读取到末尾则返回-1
int read(char[] cbuf) 从输入流中读满整个数组cbuf
String readLine() 读取一行字符串并返回，返回null表示读取到末尾
void close() 关闭流对象并释放所有资源

3.9、PrintStream类（重点）

基本概念：方便打印各种数据内容

方法声明 功能介绍
PrintStream(OutputStream out) 根据参数指定的引用构造对象
void print(String s) 用于将参数指定的内容打印出来
void pintln(String s) 用于打印字符串后并终止该行
void flush() 刷新流
void close() 关闭流对象并释放所有资源

3.10、PrintWriter类（重点）

基本概念：将对象的格式化形式打印到文本输流

方法声明 功能介绍
PrintWriter(Writer) 根据参数指定的引用构造对象
void print(String s) 用于将参数指定的内容打印出来
void pintln(String s) 用于打印字符串后并终止该行
void flush() 刷新流
void close() 关闭流对象并释放所有资源

3.11、DataOutputStream类（重点）

基本概念：主要用于以适当的方式将基本数据类型写入到输出流中

方法声明 功能介绍
DataOutputStream(OutputStream out) 根据参数指定的引用构造对象，OutputStream 是一个抽象类，实参需要传递子类对象
void write(int v) 用于将参数指定的整数一次性写入输出流，有些写入高字节
void write( ) 关闭流对象，并释放所有资源

3.12、DataInputStream类（重点）

基本概念：主要用于从输入流中读取基本类型的数据

方法声明 功能介绍
DataInputStream(InputStream in ) 根据参数指定的引用构造对象，InputStream 是一个抽象类，实参需要传递子类对象
void readInt() 用于从输入流中一次读取一个整数数据并返回
void close() 关闭流对象，并释放所有资源

3.13、ObjectOutputStream类（重点）

  • 基本概念：主要用于将一个对象的所有内容整体写到输出流中

  • 只能支持Java.io.Serializable 接口对象写入流中

  • 类通过Java.io.Serializable接口以启用序列化功能

  • 所谓序列化主要指将一个对象需要存储的相关信息有效组织成字节序列的转化过程

方法声明 功能介绍
ObjectOutputStream (OutputStream out) 根据参数指定的引用构造对象
void writeObject(Object obj) 用于将参数指定的对象整体写入到输出流中
void close() 关闭输出流并释放有关资源

3.14、ObjectInputStream类（重点）

  • 基本概念：主要用于从输入流中一次性将对象整体读出来
  • 所谓反序列化指将有效组织的字节序列恢复为一个对象及有关信息的转化过程
方法声明 功能介绍
ObjectInputStream(InputStream in ) 根据参数指定的引用构造对象
Object readObject() 主要用于从输入流中读取一个对象并返回，无法通过返回对象确定是否读到文件末尾
void close() 关闭输出流并释放有关资源

3.15、RandomAccessFile类（重点）

基本概念：支持对随机访问文件的读写操作

方法声明 功能介绍
RandomAccessFile(String name , String mode) 根据参数指定的名称和模式构对象
r：以只读方式打开
rw：打开用来读取和写入
rwd：打开用来读取和写入，同步文件内容的更新
rws：打开用来读取和写入，同步文件内容的和元数据的更新
int read() 读取单个字节的数据
void seek(long pos) 用于设置从此文件的开头开始测量的文件指针偏移量
void write(int b) 将参数指定的单个字节写入
void close() 关闭流对象释放有关资源
